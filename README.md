## week 3

### woensdag 11 maart
wat heb ik vandaag gedaan? -> Vandaag heb ik de workshop container style queries gevolgd bij Sanne. Ik heb daarna mijn tweede thema toegevoegd (een klaver koning in plaats van de harten koningin) en heb mijn controls geschreven als container style queries. Ik heb mijn radio buttons voor gezichtsronding omgezet in een slider. Ik heb ook controls toegevoegd. Door checkboxes te combineren zijn de resultaten verschillend voor de twee karakters. Koningin: bloem kan ronddraaien en hart kan breken. Koning: zwaard kan zwaaien en klaver kan dwarrelen. Beide karakters kunnen heen en weer bewegen met een control die voor beide geld. Vlak voor de checkout heb ik een achtergrond achter de kaarten toegevoegd met QUEEN QUEEN QUEEN of KING KING KING. Dit moet echter nog wel geperfectionaliseerd worden. Tijdens de checkout heb ik dit alles besproken met Alex. 

hoelang heb ik daarover gedaan? -> Ik ben hier de hele dag mee bezig geweest.

wat heb ik geleerd? -> Ik heb voor het eerst met container style queries gewerkt en daar veel over geleerd.

wat ga ik de volgende keer doen? -> Morgen ga ik de achtergrond fixen, beginnen aan de styling van het control panel en wat doen aan de positionering van de kaarten. Het idee is dat de control panel kaart grotendeels verscholen is achter de speelkaart en op hover in beeld komt.

## week 4

### woensdag 18 maart
wat heb ik vandaag gedaan? -> Vandaag ben ik begonnen met het toevoegen van een uitgebreide easter egg. Als de gezicht en nacht slider op max value staan verschijnt er een nieuwe slider die geleidelijk chaos brengt over de hele website. Ook voegde ik 3 kleinere easter eggs toe: de mond is klikbaar, er is een toetsenbord combinatie die voor een animatie zorgt (ik ga m niet helemaal verklappen... maar je zult in ieder geval de letter e nodig hebben) en ik voegde een verborgen knop toe erg op de snelweg in het midden :). Daarna ben ik begonnen met het ontclassen van mijn html. Ik realiseerde mij dat als ik dat niet slim aanpakte, het een enorme onoverzichtelijke chaos zou worden. Ik heb om te beginnnen een lijst opgezocht op mdn met html elementen en daarin gekeken welke elementen handig waren om te gebruiken. Vervolgens ben ik mijn hele html afgegaan en heb ik de divs 1 voor 1 vervangen. Ik dacht dat het handig zou zijn om veel verschillende elementen te gebruiken zodat ik niet alleen maar bezig was div nth-of-type(64)... haha. Tijdens dit hele proces heb ik wel de data-naam attributen laten staan zodat ik nog kon zien wat wat was. Toen de html aangepast was ben ik begonnen met de hele css langslopen en de selectoren 1 voor 1 aanpassen. Ondertussen controleerde ik continue of alles nog werkte na een aanpassing. Ik probeerde hier zo snel mogelijk doorheen te gaan maar dit kreeg ik vandaag natuurlijk niet af hahaha. Tijdens de checkout heb ik dit alles besproken met Rafi.

hoelang heb ik daarover gedaan? -> Ik ben hier vandaag van half 10 tot 4 mee bezig geweest.

wat heb ik geleerd? -> Het enige nieuwe dat ik vandaag heb geleerd is hoe ik een keyboard animation kan toevoegen.

wat ga ik morgen doen? -> Morgen moet ik voor 18:00 mijn werk inleveren en mijn to do list is als volgt: ontclassen afmaken en typografie verbeteren (variable font toevoegen).


### donderdag 19 maart
wat heb ik vandaag gedaan? -> Vandaag ben ik verder gegaan met het ontclassen. Ik heb gisteren goed kunnen inkomen dus vandaag ging het gelukkig een stuk sneller. Daarna heb ik (met wat hulp van Sabrina) uitgezocht hoe een variable font werkt, hier had ik namelijk nooit eerder mee gewerkt. Toen ik ook dat had toegevoegd heb ik mijn ReadMe geupdate.

hoelang heb ik daarover gedaan? -> Het ontclassen had ik na 3 uurtjes doorwerken af. Het variable font ben ik een uurtje mee bezig geweest.

wat heb ik geleerd? -> Ik heb geleerd hoe ik een variable font moet gebruiken.

wat ga ik morgen doen? -> Morgen heb ik mijn eindgesprek met Nils :)



### weekvoortgang
Deze laatste week voelde als de grote finale van het project. Waar ik in de eerste weken nog worstelde met simpele positionering, is het nu gekomen tot een zorgvuldig opgebouwt geheel. De grootste uitdaging (en overwinning) was  het "ontclassen" van de html. De css werd daardoor een flinke puzzel met uitgebreide selectoren. Ik  vond het leerzaam om hier veel mee bezig te zijn, maar het was wel een heel werk. Daarnaast heb ik mij deze week gestort op de details van de animaties en natuurlijk de easter eggs. Ik vond dit een heel leuk project om mee bezig te zijn en om te ontdekken hoeveel er wel niet mogelijk is met alleen css! Ik ben trots op mijn eindresultaat :)
